Версия 0.5
1. При обращении к не тому элементу программа должна показать человекочитабельную ошибку.
2. Сортировка должна запускаться на имеющемся буфере (нельзя передвигать элементы). Для этого напишите свой итератор, вам пригодятся следующие вещи:
    using iterator_category
    using value_type
    using difference_type
    using pointer
    using reference
